# React Minimal Playground

このレポジトリは、React (Vite + TypeScript) を使用した最小構成の個人用プレイグラウンドです。
AI エージェント（Claude Code / Gemini CLI 等）がプロジェクトの意図を理解するためのガイドラインです。

## 1. プロジェクト概要
- **目的**: 新しい技術の実験、UIコンポーネントの試作、小規模なツールの開発。
- **スタック**: React, TypeScript, Vite.
- **方針**: 複雑なフォルダ階層を避け、`src/` 直下で迅速に開発を行う。

## 2. ディレクトリ構成
AI は以下の構造を前提にコードを生成・修正してください。

- `index.html`: エントリポイント。
- `src/main.tsx`: React のマウント設定。基本的には変更不要。
- `src/App.tsx`: **メインの実験場**。ここにコードを直接記述するか、新しいコンポーネントを呼び出す。
- `src/components/`: 必要に応じて作成するコンポーネント用フォルダ。
- `vite.config.ts`: Vite の設定ファイル。

## 3. 開発コマンド
- `npm run dev`: 開発サーバーの起動。
- `npm run build`: ビルドの実行。
- `npm run preview`: ビルド成果物のローカル確認。

## 4. AI への指示（コンテキスト）
- コードを生成する際は、型安全のために TypeScript を使用してください。
- スタイルに関しては、インラインスタイルまたは `App.css` を使用してください。
- 新しい機能を追加する際は、まず `App.tsx` を修正し、肥大化してきたら `src/components/` に切り出してください。
